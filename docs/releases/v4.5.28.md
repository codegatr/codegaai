# CODEGA AI v4.5.28 Release Notes

## Phoenix Sprint 2

This release continues the Phoenix recovery track with local model resilience improvements.

### Changed

- Bumped the desktop package version to `4.5.28`.
- Reduced the local Ollama chat timeout baseline from 45 seconds to 35 seconds.
- Reordered fallback models to prefer lightweight models first.
- Added Phoenix Sprint 2 release gates to `npm run check`.
- Added Sprint 2 documentation and acceptance checklist.

### Why this matters

If the selected model is slow or stuck, CODEGA AI should recover faster instead of appearing locked on `Dusunuyorum...`.

### Validation

Run from `apps/codegaai-desktop`:

```bash
npm run check
```

Manual smoke checks:

1. `PHP nedir? Tek cumle.` should return without indefinite waiting.
2. `Yapay zeka nedir? Kisa acikla.` should return without indefinite waiting.
3. A heavier planning prompt should either answer or fail clearly instead of returning an empty response.
