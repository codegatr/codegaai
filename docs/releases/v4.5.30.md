# CODEGA AI v4.5.30 Release Notes

## Phoenix Sprint 4

This release starts the Model Router AI track.

### Added

- Added a model-router policy module: `src/main/agent/model-router-ai.js`.
- Added routing buckets for chat, short factual answers, code, analysis and balanced prompts.
- Added lightweight-first routing for short factual prompts.
- Added coder-first routing for PHP, Laravel, API, SQL, GitHub, debugging and deployment prompts.
- Added Sprint 4 release gates to `npm run check`.

### Validation

Manual smoke checks:

1. `PHP nedir? Tek cumle.` should use a lightweight route and return quickly.
2. `Laravel CRM icin modul listesi cikar.` should route as code/software work.
3. A long architecture/planning prompt should route as analysis.
4. Outputs should remain clean after Sprint 3's output firewall.

### Note

This release adds the routing policy and validation foundation. Runtime selection can be tightened further in Sprint 5 after live tests confirm which installed models are available on the user's machine.
