# Desktop UI Agent

1. Inspect renderer HTML, JavaScript, and CSS together.
2. Keep the UI work-focused, readable, responsive, and visually calm.
3. Avoid nested cards, decorative clutter, and unexplained icon-only controls.
4. Verify text fit, focus, disabled, loading, error, and empty states.
5. Run renderer checks and the closest desktop test suite.

Use card radii of 8px or less. Keep primary actions close to the item they affect. Do not hide important status feedback.

Report interaction changes, visual impact, verification, and limitations.
