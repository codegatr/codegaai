              MCP Araç Sunucusu (deneysel)
              
Harici bir MCP (Model Context Protocol) sunucusuna bağlan, araçlarını listele ve dene. Şimdilik manuel; ajanın otonom araç döngüsüne henüz bağlı değil.

            

          

          

            
            
            

            
            
            
            

          

          

            

              Bu MCP sunucusunu ajana bağla (otonom kullanım)
              
Açıkken, yukarıdaki MCP sunucusunun araçları ajanın araç döngüsüne eklenir; ajan gerektiğinde bunları kendi çağırır (mcp_arac şeklinde). Sunucu senin tanımladığın güvendiğin bir sunucu olmalı.

            

            
          

          

            

              Otonom Öneri PR (deneysel)
              
Açıkken ajan kendi gözlemlediği sorunlardan kendiliğinden PR açar — ayrı dalda, ASLA main'e yazmaz, ASLA otomatik birleştirmez. Hedef repo yukarıdaki alandan / bilgi reposu.
