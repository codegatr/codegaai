              Sürekli Öğrenme
              
Açıkken, belirlediğin konuları periyodik araştırıp bilgi toplar (her ~25 dk, tur başına 1 konu). Kaynaklar korumalı çalışır; biri başarısız olursa döngü devam eder. Uygulama açık olmalı.

            

            
          

          

            
            
            
            
            

            
          

          

            

              Anlamsal Arama (embeddings)
              
Öğrenilen bilgiyi anahtar-kelime yerine anlam benzerliğiyle eşler — cevaplar daha isabetli olur. Gerekir: terminalde ollama pull nomic-embed-text. Kapalıysa anahtar-kelime kullanılır.

            

            
          

          

            

              Notları Damıt (model ile özetle)
              
Açıkken, öğrenilen ham kaynak notlarını model kısa kalıcı bir özete indirir; depoya "özet" olarak eklenir. Ollama/model açık olmalı. Yavaş donanımda yük getirebilir.

              MCP Araç Sunucusu (deneysel)
              
Harici bir MCP (Model Context Protocol) sunucusuna bağlan, araçlarını listele, dene ve isteğe bağlı olarak ajanın otonom araç döngüsüne ekle.

            

          

          

            
            
            

            
            
            
            

          

          

            

              Bu MCP sunucusunu ajana bağla (otonom kullanım)
              
Açıkken, yukarıdaki MCP sunucusunun araçları ajanın araç döngüsüne eklenir; ajan gerektiğinde bunları kendi çağırır (mcp_arac şeklinde). Sunucu senin tanımladığın güvendiğin bir sunucu olmalı.

            

            
          

          

            

              Otonom Öneri PR (deneysel)
              
Açıkken ajan kendi gözlemlediği sorunlardan kendiliğinden PR açar — ayrı dalda, ASLA main'e yazmaz, ASLA otomatik birleştirmez. Hedef repo yukarıdaki alandan / bilgi reposu.
